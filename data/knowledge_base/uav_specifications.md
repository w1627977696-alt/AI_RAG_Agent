# 无人机技术规格与维护手册

## 1. 无人机硬件规格

### 1.1 机架与动力系统
- **机架类型**：四旋翼X型布局
- **对角轴距**：450mm
- **最大起飞重量**：2.5kg
- **电机型号**：2212 KV920无刷电机
- **螺旋桨**：10x4.5碳纤维螺旋桨
- **正常工作RPM范围**：4000-6000 RPM
- **电机工作温度范围**：-10°C ~ 80°C

### 1.2 电池系统
- **电池类型**：4S LiPo 5000mAh
- **标称电压**：14.8V
- **满充电压**：16.8V
- **截止电压**：12.0V（单片3.0V）
- **最大放电倍率**：30C
- **工作温度范围**：-10°C ~ 45°C
- **循环寿命**：约500次（80%容量保持率）
- **安全电量阈值**：15%（告警），10%（强制返航），5%（紧急着陆）

### 1.3 传感器系统
- **GPS模块**：u-blox M8N，精度2.5m CEP
- **IMU**：InvenSense MPU-6000，6轴
- **气压计**：MS5611，高度精度±1m
- **磁罗盘**：HMC5883L
- **测距传感器**：TFmini-S激光测距，0.1-12m
- **温度传感器**：NTC 10K，精度±0.5°C

### 1.4 通信系统
- **数据链路**：2.4GHz数字图传
- **遥控链路**：900MHz长距离遥控
- **最大通信距离**：10km（无遮挡）
- **正常信号强度范围**：-30dBm ~ -70dBm
- **信号告警阈值**：-80dBm
- **信号丢失阈值**：-90dBm

## 2. 飞行性能参数

### 2.1 飞行能力
- **最大飞行速度**：20m/s
- **最大爬升速度**：5m/s
- **最大下降速度**：3m/s
- **最大飞行高度**：500m（相对高度）
- **最大飞行时间**：35分钟（无载荷）
- **抗风能力**：6级风

### 2.2 正常飞行参数范围
| 参数 | 正常范围 | 告警范围 | 危险范围 |
|------|----------|----------|----------|
| 温度 | 20-45°C | 45-65°C | >65°C 或 <-10°C |
| 振动 | 0-3g | 3-7g | >7g |
| 电量 | 30-100% | 15-30% | <15% |
| 信号强度 | -30~-70dBm | -70~-85dBm | <-85dBm |
| 高度 | 10-400m | 400-500m | >500m 或 <5m |
| 速度 | 0-15m/s | 15-20m/s | >20m/s |

## 3. 集群协同规范

### 3.1 编队间距要求
- **最小安全间距**：水平5m，垂直3m
- **标准编队间距**：水平10m，垂直5m
- **编队变换缓冲距离**：水平15m
- **紧急避障距离**：3m

### 3.2 集群通信协议
- **心跳间隔**：1秒
- **位置广播间隔**：0.5秒
- **失联判定时间**：10秒无心跳
- **集群重组触发条件**：失联节点≥2或编队完整性<80%

### 3.3 集群任务类型
1. **区域巡检**：网格化覆盖，每架UAV负责指定区域
2. **编队飞行**：保持固定队形飞行，用于表演或侦察
3. **搜索救援**：扇形展开搜索，实时共享发现信息
4. **物流配送**：点对点飞行，动态避障和路径规划

## 4. 维护保养计划

### 4.1 日常维护（每次飞行后）
- 检查螺旋桨是否有裂纹或变形
- 清理机架上的灰尘和碎屑
- 检查电池外观是否有鼓包
- 记录飞行日志

### 4.2 周度维护
- 校准IMU和磁罗盘
- 检查电机轴承松动情况
- 测试所有传感器读数准确性
- 更新固件（如有）

### 4.3 月度维护
- 更换磨损螺旋桨
- 深度清理电机（压缩空气）
- 电池健康度检测
- 通信系统全面测试
- 机架结构完整性检查
